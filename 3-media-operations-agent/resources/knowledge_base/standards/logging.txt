# Video Content Compliance Standards for Internal Video Logging
**Document Number:** LOG-VID-001  
**Last Updated:** March 2025  
**Department:** Content Operations

## 1. CONTENT TYPE REQUIREMENTS

### 1.1 News Content
- **Header Requirements:**
  - Short headline of news event
  - Key organization/individual involved
  - Date of event
- **Description Requirements:**
  - Maximum 200 words
  - Include 5W's (Who, What, Where, When, Why)
  - Event significance and impact
  - Source attribution

### 1.2 Film Content
- **Header Requirements:**
  - Film title and release year
  - Production company
  - Content type identifier
- **Description Requirements:**
  - Maximum 200 words
  - Cast members and roles
  - Director and key crew
  - Plot synopsis
  - Production details

### 1.3 Sports Content
- **Header Requirements:**
  - Event name
  - Teams/Athletes involved
  - Venue name
- **Description Requirements:**
  - Maximum 200 words
  - Match/Game details
  - Key highlights
  - Final outcome
  - Significant statistics

## 2. DISTRIBUTION CHANNEL REQUIREMENTS
- Title: Maximum 120 characters
- Description: Maximum 200 words
- Minimum 3 tags required
- Voice: Professional and informative
- All titles must use title case
- Video Type: News, Film, or Sports classification
- Location: City and country
- Venue: Specific location name
- Resolution: Available quality options
- Frame Rate: Captured speed
- Encoding: Format specification
- Missing data fields marked as "N/A"

## 3. DATA MODELS
```json
{
    "header": "[short description of the video]",
    "description": "[content summary about 200 words]",
    "video_type": "[Film or Sports]",
    "location": "[city]",
    "venue": "[venue name]",
    "tags": [“relevant”, “tags”, “3 minimum”],
    "technical_specs": {
        "resolution": "1280x720",
        "frame_rate": "30",
        "encoding": "h264",
    }
}
```
