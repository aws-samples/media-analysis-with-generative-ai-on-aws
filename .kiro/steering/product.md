# Video Understanding with Generative AI on AWS Workshop

This is a comprehensive hands-on workshop for developers and solution builders focused on implementing video understanding use cases using AWS AI Services and foundation models through Amazon Bedrock.

## Core Capabilities

The workshop demonstrates video analysis techniques across four main areas:

1. **Media Analysis using Bedrock Data Automation (BDA)** - Automated video content extraction and analysis
2. **Media Analysis using Amazon Nova** - Foundation model-based video understanding with visual and audio segmentation
3. **Media Operations Agents** - AI agents for sports analysis, compliance checking, and content operations using Strands Agent SDK and Bedrock AgentCore
4. **Live Media Analysis Agent** - Real-time streaming video analysis and companion agents

## Key Use Cases

- Video time segmentation and scene detection
- Ad break detection and contextual advertising
- Video summarization and content generation
- Multi-modal semantic search (text + visual)
- Sports event analysis and player identification
- Compliance monitoring and content standards validation
- Live streaming companion experiences

## Target Audience

Developers, solution architects, and media professionals working with video content analysis, AI-powered media operations, and generative AI applications in the media & entertainment industry.
