# Video Content Compliance Standards for Social Media
**Document Number:** SOC-MED-001  
**Last Updated:** March 2025  
**Department:** Content Operations

## 1. CONTENT TYPE REQUIREMENTS

### 1.1 News Content
- **Required Elements (5W's):**
  - Who: Key individuals/organizations involved
  - What: Main event/incident description
  - Where: Specific location
  - When: Date and time
  - Why: Context of the event

### 1.2 Film Content
- **Required Elements:**
  - Movie title with release year
  - Main cast members with @mentions
  - Character names

### 1.3 Sports Content
- **Required Elements:**
  - Event/Game name
  - Venue information
  - Team names with @handles
  - Key players mentioned

## 2. DISTRIBUTION CHANNEL REQUIREMENTS
- Description: Maximum 75 words
- Hashtags: 2-3 relevant tags required
- Mentions: Include applicable @handles
- Tone: Casual and engaging
- Emoji: Strategic placement for engagement
- Structure: Hook, Main Point, Call-to-action
